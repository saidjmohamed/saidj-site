## 🚀 التثبيت والتشغيل

### **المتطلبات المسبقة**
- 📦 **Node.js** (الإصدار 18 أو أحدث)
- 📋 **npm** أو **yarn** أو **pnpm**

### **خطوات التثبيت**

```bash
# استنسخ المشروع
git clone https://github.com/saidjmohamed/saidj-site.git

# انتقل إلى مجلد المشروع
cd saidj-site

# ثبت التبعيات
npm install

# انشئ ملف البيئة
cp .env.example .env.local

# أضف مفتاح Gemini API
echo "GEMINI_API_KEY=your_api_key_here" >> .env.local
```

### **تشغيل المشروع**

```bash
# تشغيل الخادم المحلي
npm run dev

# بناء المشروع للإنتاج
npm run build

# معاينة البناء
npm run preview
```
